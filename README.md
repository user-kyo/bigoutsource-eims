# Employee Information Management System (EIMS)

A modern, full-stack Employee Information Management System built with a React + Vite frontend, a Node.js + Express backend, and powered by Prisma and PostgreSQL.

---

## 🚀 Tech Stack

- **Frontend:** React, Vite, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express.js, Prisma (ORM)
- **Database:** PostgreSQL (via Docker or Supabase)
- **Containerization:** Docker & Docker Compose

---

## 📂 Project Structure
